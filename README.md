## 🚀 What is KisanKonnect?

**KisanKonnect** is a futuristic, AI-powered super app designed for Indian farmers. It combines the power of voice AI, real-time data, and hyperlocal intelligence to solve real problems faced by 600M+ farmers across Bharat.

> Think: **ChatGPT + WhatsApp + Google Maps — built for Indian farmers.**

No English. No typing. Just speak.

---

## ✨ Key Features

### 🎙️ Voice-First Community Feed
- Record crop problems in Hindi, Bhojpuri, Maithili, Punjabi
- AI instantly analyses voice notes
- Community of farmers help each other
- Voice replies and upvoting system

### 🤖 KisanAI Crop Advisory
- AI-powered crop disease diagnosis
- 94%+ confidence detection
- Step-by-step treatment plans in Hindi
- Voice playback of advisory

### 📈 Live Mandi Price Tracker
- Real-time prices from 5 nearby mandis
- Price trend indicators (↑↓)
- AI recommendation: "Best time to sell"
- Filter by crop, district, state

### 🌦️ Hyperlocal Weather + Crop Calendar
- 7-day weather forecast
- Rain alerts, heatwave warnings
- Crop-specific recommendations
- Sowing & harvesting reminders

### 🚨 SOS Crop Emergency System
- One-tap emergency button
- Voice recording for emergency
- Instant alert to nearby farmers
- AI emergency advisory

### 📸 AI Crop Photo Diagnosis
- Upload or capture crop photo
- AI scans for disease in seconds
- Shows disease name, severity, treatment
- Voice response in Hindi

### 📡 Offline Mode + Smart Sync
- Works on 2G networks
- Saves data locally when offline
- Auto-syncs when internet returns
- Offline advisory, schemes, notes

### 🏅 Farmer Trust Score System
- Community reputation engine
- Verified Kisan Expert badges
- Leaderboard and gamification
- Contribution levels

### 💬 WhatsApp-First Onboarding
- Send "KISAN" to get started
- Full onboarding via WhatsApp chat
- Mandi prices, weather, advisory on WhatsApp

### 🧾 AI Scheme Eligibility Engine
- PM-KISAN, Fasal Bima, KCC, Soil Health Card
- Auto-suggests schemes based on your profile
- Eligibility % and document checklist
- One-click apply simulation

## 🚀 How to Run

1. Clone this repository:
```bash
git clone https://github.com/asthamosmi-arch/KisanKonnect.git
```

2. Open `KisanKonnect_v2.html` in any browser

3. No server needed — runs entirely in the browser!
